## Setup Environment - Anaconda

## Setup Environment - Shell/Terminal
```

cd Analisis_data_dicoding
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run steamlit app
```

streamlit run dashboard/dashboard.py
```
